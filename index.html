3. State Machine Logic

The core engine follows a state machine where the current "Tier" (T0 through T4) dictates the portfolio mix.

  • Daily Transition: Moves between states freely based on current drawdown.
  • Ratchet Transition: Moves to higher tiers (deeper DD) but refuses to move back to lower tiers until a 0% drawdown (New All-Time High) is reached.
