📘 Technical Strategy Specifications

1. Overview & Data Source

This tool compares 19 strategy variants — ranging from our optimized Hall of Fame (Beast, Scalpel, Shield) to the new 1x–4x Ladder standard. The core thesis: Aggressively buy the dip using logarithmic leverage scaling, and protect capital using trend-filters and safety buffers. All simulations are "Honest" backtests—meaning they use strictly lagged data (Shift-1) to eliminate look-ahead bias and reflect achievable real-world performance. Simulations begin on July 1, 2002, capturing the 2008 GFC, the 2020 COVID crash, and the 2022 inflationary bear market.

All returns are Total Return (dividends and interest reinvested). The data is sourced from Yahoo Finance using the following proxies:

Leveraged returns are simulated as: Return = (Leverage × SPY) − ((Leverage − 1) × VFISX). Inflation data is fetched monthly from the St. Louis Fed and interpolated daily.

2. Asset Allocation Tier Matrices

Each strategy dynamically shifts its portfolio weights based on the previous day's drawdown (distance from All-Time High). This prevents look-ahead bias — you only act on yesterday's information.

3. Rebalancing Rules

FrequencyHow It WorksTrade-Off
DailyRecalculates drawdown and shifts weights every business day.Most responsive, highest transaction cost.
MonthlySamples signal on the last trading day. Locks for the next month.Lower costs, can miss intra-month moves.
RatchetLocks the highest tier reached. Never de-leverages until a new ATH.Max recovery capture, stays leveraged in false bounces.

4. Risk Controls

Trend Filter (200-day SMA)

If the S&P 500 closes below its 200-day SMA, all strategies revert to the Safe Tier (T0). This overrides both drawdown tiers and ratchet lock-in.

Look-Ahead Bias Prevention

All signals use the previous day's close via shift(1). Every trade decision uses only information available at the time.
